# Reti-Sociali-TSS
Progetto svolto per l'esame di Reti Sociali dell'Università degli Studi di Salerno, A.A. 2022-2023.

Implementazione dell'algoritmo TSS per effettuare Target Set Selection su una rete, presentato nell'articolo "Discovering Small Target Sets in Social Networks: A Fast and Effective Algorithm", di G. Cordasco, L. Gargano, M.Mecchia, A. Rescigno e U. Vaccaro, 2018, Algorithmica.

Il dataset è disponibile a https://networkrepository.com/socfb-MIT.php
